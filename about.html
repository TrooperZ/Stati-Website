About Stati

FAQ

What is Stati?

Stati is a game statistics bot that displays your kills, wins, deaths, and other factors in a game.

How does stati get the data?

The games we support have an API that we can use to fetch information. This data is then returned in JSON format and we parse it and output it in an embed format.

I can 't see the output from stati!

Please update your client, make sure Stati has Embed Links permissions, and enable embeds in your Discord settings.

Stati says my name is invalid!

If the name has spaces, put it in quotes ("this is a name "). If you cannot use your Steam name, use steamid.io to find your Steam ID. For Ubisoft games, use your display name.

What data does stati collect?

Stati only collects the following:
- User ID for premium purchases
- Channel ID and message ID for refreshing messages
- How often a command is used
- Server and member count

Is this against any game TOS?

Nope! All of the data is provided by the developers themselves and is not against any TOS.
